# 🏦 P2P LENDING PROJECT - COMPREHENSIVE DOCUMENTATION

## 📋 TỔNG QUAN DỰ ÁN

Dự án P2P Lending là một nền tảng cho vay ngang hàng hiện đại tại Việt Nam, được thiết kế để kết nối người vay và người cho vay thông qua công nghệ số. Dự án này bao gồm đầy đủ các thành phần từ thiết kế database, công nghệ, DevOps pipeline đến các checklist thực thi chi tiết.
